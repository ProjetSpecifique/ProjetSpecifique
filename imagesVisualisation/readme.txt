-Modifier l'attribut 'imagesPath' dans le fichier scripts/script.js avec le nom du dossier.
-Lancer index.js dans un navigateur.

 colonne 2: 0 ou 1 pour image dans ou hors catégorie
 colonne n: tags de l'images
